1. Introdução

Este projeto tem como objetivo demonstrar a implementação de técnicas de segurança em uma comunicação entre duas máquinas virtuais (VMs). Foram abordados os seguintes cenários:

  • Comunicação usando criptografia simétrica (AES).
  • Comunicação usando criptografia assimétrica (RSA).
  • Troca de chaves AES usando RSA e comunicação com AES.
  • Comunicação usando funções de hash (SHA-256) para integridade.
  • Comunicação com confidencialidade (AES) e integridade (SHA-256).

2. Configuração do Ambiente

Requisitos

  • VirtualBox: Para criar e gerenciar as máquinas virtuais.
  • Sistema Operacional: Ubuntu (ou outra distribuição Linux) instalado nas VMs.
  • Bibliotecas Python:
    • pycryptodome: Para criptografia e funções de hash.
    • socket: Para comunicação em rede.
    • threading: Para envio e recebimento simultâneo de mensagens.

Configuração das VMs

  1. Crie duas VMs no VirtualBox.
  2. Configure as VMs em modo bridge para que possam se comunicar na mesma rede.
  3. Defina IPs fixos para as VMs usando o Netplan: 
    sudo netplan apply
  4. Instale as dependências Python nas VMs: 
    sudo apt update
sudo apt install python3 python3-pip
pip3 install pycryptodome

3. Cenários Implementados

Cenário 1: Comunicação com Criptografia Simétrica (AES)

Objetivo: Demonstrar a comunicação segura usando criptografia simétrica com o algoritmo AES.

Funcionamento:

  • O servidor e o cliente compartilham a mesma chave AES (KEY) e vetor de inicialização (IV).
  • As mensagens são criptografadas e descriptografadas usando AES no modo CBC.

Códigos:

Servidor (VM1)

from Crypto.Cipher import AES
import socket
import threading

# Chave e IV devem ser os mesmos no cliente e no servidor
KEY = b'Sixteen byte key'  # 16 bytes
IV = b'Sixteen byte iv '   # 16 bytes

def encrypt_message(message):
    cipher = AES.new(KEY, AES.MODE_CBC, IV)
    padding_length = 16 - (len(message) % 16)
    padded_message = message.encode('utf-8') + bytes([padding_length]) * padding_length
    return cipher.encrypt(padded_message)

def decrypt_message(encrypted_message):
    cipher = AES.new(KEY, AES.MODE_CBC, IV)
    decrypted = cipher.decrypt(encrypted_message)
    padding_length = decrypted[-1]
    return decrypted[:-padding_length].decode('utf-8')

def receive_messages(client_socket):
    while True:
        try:
            encrypted_message = client_socket.recv(1024)
            if not encrypted_message:
                break
            decrypted_message = decrypt_message(encrypted_message)
            print(f"\nMensagem recebida: {decrypted_message}")
        except Exception as e:
            print(f"\nErro ao receber mensagem: {e}")
            break

def send_messages(client_socket):
    while True:
        try:
            message = input("\nDigite uma mensagem para enviar: ")
            encrypted_message = encrypt_message(message)
            client_socket.send(encrypted_message)
        except Exception as e:
            print(f"\nErro ao enviar mensagem: {e}")
            break

def start_server():
    server = socket.socket(socket.AF_INET, socket.SOCK_STREAM)
    server.bind(('192.168.1.100', 5000))
    server.listen(1)
    print("\nServidor TCP aguardando conexões na porta 5000...")

    client_socket, addr = server.accept()
    print(f"\nConexão estabelecida com {addr}")

    threading.Thread(target=receive_messages, args=(client_socket,)).start()
    threading.Thread(target=send_messages, args=(client_socket,)).start()

if __name__ == "__main__":
    start_server()

Cenário 2: Comunicação com Criptografia Assimétrica (RSA)

Objetivo: Demonstrar a comunicação segura usando criptografia assimétrica com o algoritmo RSA.

Funcionamento:

  • O servidor e o cliente geram seus próprios pares de chaves RSA (pública e privada).
  • As mensagens são criptografadas com a chave pública do destinatário e descriptografadas com a chave privada do receptor.

Códigos:

Servidor (VM1)

from Crypto.PublicKey import RSA
from Crypto.Cipher import PKCS1_OAEP
import socket
import threading

# Gera um par de chaves RSA para o servidor
key_pair_server = RSA.generate(2048)
private_key_server = key_pair_server.export_key()
public_key_server = key_pair_server.publickey().export_key()

# Variável para armazenar a chave pública do cliente
public_key_client = None

def encrypt_message(message, public_key):
    cipher = PKCS1_OAEP.new(RSA.import_key(public_key))
    return cipher.encrypt(message.encode('utf-8'))

def decrypt_message(encrypted_message):
    cipher = PKCS1_OAEP.new(RSA.import_key(private_key_server))
    return cipher.decrypt(encrypted_message).decode('utf-8')

def receive_messages(client_socket):
    global public_key_client
    try:
        # Recebe a chave pública do cliente
        public_key_client = client_socket.recv(4096)
        print("\nChave pública do cliente recebida.")

        # Envia a chave pública do servidor para o cliente
        client_socket.send(public_key_server)
        print("Chave pública do servidor enviada.")

        # Agora pode receber mensagens criptografadas com RSA
        while True:
            encrypted_message = client_socket.recv(4096)
            if not encrypted_message:
                break
            decrypted_message = decrypt_message(encrypted_message)
            print(f"\nMensagem recebida: {decrypted_message}")
    except Exception as e:
        print(f"\nErro ao receber mensagem: {e}")
    finally:
        client_socket.close()

def send_messages(client_socket):
    try:
        while True:
            message = input("\nDigite uma mensagem para enviar: ")
            if public_key_client:
                encrypted_message = encrypt_message(message, public_key_client)
                client_socket.send(encrypted_message)
            else:
                print("\nAguardando chave pública do cliente...")
    except Exception as e:
        print(f"\nErro ao enviar mensagem: {e}")

def start_server():
    server = socket.socket(socket.AF_INET, socket.SOCK_STREAM)
    server.bind(('192.168.1.100', 5000))
    server.listen(1)
    print("\nServidor TCP aguardando conexões na porta 5000...")

    client_socket, addr = server.accept()
    print(f"\nConexão estabelecida com {addr}")

    threading.Thread(target=receive_messages, args=(client_socket,)).start()
    threading.Thread(target=send_messages, args=(client_socket,)).start()

if __name__ == "__main__":
    start_server()

Cenário 3: Troca de Chaves AES usando RSA e Comunicação com AES

Objetivo: Usar criptografia assimétrica (RSA) para trocar uma chave simétrica (AES) e, em seguida, usar AES para comunicação.

Funcionamento:

  • O cliente gera uma chave AES e a criptografa com a chave pública do servidor.
  • O servidor descriptografa a chave AES com sua chave privada.
  • A partir daí, a comunicação usa AES para criptografar os dados.

Códigos:

Servidor (VM1)

from Crypto.PublicKey import RSA
from Crypto.Cipher import PKCS1_OAEP, AES
from Crypto.Random import get_random_bytes
import socket
import threading

# Gera um par de chaves RSA para o servidor
key_pair_server = RSA.generate(2048)
private_key_server = key_pair_server.export_key()
public_key_server = key_pair_server.publickey().export_key()

# Variável para armazenar a chave simétrica (AES)
aes_key = None

def decrypt_aes_key(encrypted_aes_key):
    cipher = PKCS1_OAEP.new(RSA.import_key(private_key_server))
    return cipher.decrypt(encrypted_aes_key)

def encrypt_message(message):
    cipher = AES.new(aes_key, AES.MODE_EAX)
    nonce = cipher.nonce
    ciphertext, tag = cipher.encrypt_and_digest(message.encode('utf-8'))
    return nonce + ciphertext + tag

def decrypt_message(encrypted_message):
    nonce = encrypted_message[:16]
    ciphertext = encrypted_message[16:-16]
    tag = encrypted_message[-16:]
    cipher = AES.new(aes_key, AES.MODE_EAX, nonce=nonce)
    return cipher.decrypt_and_verify(ciphertext, tag).decode('utf-8')

def receive_messages(client_socket):
    global aes_key
    try:
        # Recebe a chave pública do cliente
        public_key_client = client_socket.recv(4096)
        print("\nChave pública do cliente recebida.")

        # Envia a chave pública do servidor para o cliente
        client_socket.send(public_key_server)
        print("Chave pública do servidor enviada.")

        # Recebe a chave AES criptografada
        encrypted_aes_key = client_socket.recv(4096)
        aes_key = decrypt_aes_key(encrypted_aes_key)
        print("Chave AES recebida e descriptografada.")

        # Agora pode receber mensagens criptografadas com AES
        while True:
            encrypted_message = client_socket.recv(4096)
            if not encrypted_message:
                break
            decrypted_message = decrypt_message(encrypted_message)
            print(f"\nMensagem recebida: {decrypted_message}")
    except Exception as e:
        print(f"\nErro ao receber mensagem: {e}")
    finally:
        client_socket.close()

def send_messages(client_socket):
    try:
        while True:
            message = input("\nDigite uma mensagem para enviar: ")
            if aes_key:
                encrypted_message = encrypt_message(message)
                client_socket.send(encrypted_message)
            else:
                print("\nAguardando chave AES...")
    except Exception as e:
        print(f"\nErro ao enviar mensagem: {e}")

def start_server():
    server = socket.socket(socket.AF_INET, socket.SOCK_STREAM)

Cenário 4: Comunicação usando Funções de Hash (SHA-256) para Integridade

Objetivo: Garantir a integridade das mensagens usando funções de hash (SHA-256).

Funcionamento:

  • Cada mensagem é enviada junto com seu hash SHA-256.
  • O receptor recalcula o hash da mensagem recebida e compara com o hash enviado para verificar a integridade.

Códigos:

Servidor (VM1)

from Crypto.Hash import SHA256
import socket
import threading

def calculate_hash(message):
    return SHA256.new(message.encode('utf-8')).hexdigest()

def receive_messages(client_socket):
    try:
        while True:
            # Recebe a mensagem e o hash
            data = client_socket.recv(4096).decode('utf-8')
            if not data:
                break

            # Separa a mensagem e o hash
            received_message, received_hash = data.split('|')

            # Calcula o hash da mensagem recebida
            calculated_hash = calculate_hash(received_message)

            # Verifica a integridade
            if received_hash == calculated_hash:
                print(f"\nMensagem recebida (íntegra): {received_message}")
            else:
                print(f"\nMensagem recebida (corrompida): {received_message}")
    except Exception as e:
        print(f"\nErro ao receber mensagem: {e}")
    finally:
        client_socket.close()

def send_messages(client_socket):
    try:
        while True:
            message = input("\nDigite uma mensagem para enviar: ")
            if message:
                # Calcula o hash da mensagem
                message_hash = calculate_hash(message)

                # Envia a mensagem e o hash
                client_socket.send(f"{message}|{message_hash}".encode('utf-8'))
    except Exception as e:
        print(f"\nErro ao enviar mensagem: {e}")

def start_server():
    server = socket.socket(socket.AF_INET, socket.SOCK_STREAM)
    server.bind(('192.168.1.100', 5000))
    server.listen(1)
    print("\nServidor TCP aguardando conexões na porta 5000...")

    client_socket, addr = server.accept()
    print(f"\nConexão estabelecida com {addr}")

    threading.Thread(target=receive_messages, args=(client_socket,)).start()
    threading.Thread(target=send_messages, args=(client_socket,)).start()

if __name__ == "__main__":
    start_server()

Cenário 5: Comunicação com Confidencialidade e Integridade

Objetivo: Unir os conceitos dos cenários 3 e 4 para garantir confidencialidade (usando AES e RSA) e integridade (usando SHA-256).

Funcionamento:

  • A chave AES é trocada de forma segura usando RSA.
  • As mensagens são criptografadas com AES.
  • O hash SHA-256 é usado para garantir que as mensagens não foram alteradas.

Códigos:

Servidor (VM1)

from Crypto.PublicKey import RSA
from Crypto.Cipher import PKCS1_OAEP, AES
from Crypto.Hash import SHA256
from Crypto.Random import get_random_bytes
import socket
import threading

# Gera um par de chaves RSA para o servidor
key_pair_server = RSA.generate(2048)
private_key_server = key_pair_server.export_key()
public_key_server = key_pair_server.publickey().export_key()

    cipher = PKCS1_OAEP.new(RSA.import_key(private_key_server))
    return cipher.decrypt(encrypted_aes_key)

def encrypt_message(message):
    cipher = AES.new(aes_key, AES.MODE_EAX)
    nonce = cipher.nonce
    ciphertext, tag = cipher.encrypt_and_digest(message.encode('utf-8'))
    return nonce + ciphertext + tag

    nonce = encrypted_message[:16]
    ciphertext = encrypted_message[16:-16]
    tag = encrypted_message[-16:]
    cipher = AES.new(aes_key, AES.MODE_EAX, nonce=nonce)
    return cipher.decrypt_and_verify(ciphertext, tag).decode('utf-8')

def calculate_hash(message):
    return SHA256.new(message.encode('utf-8')).hexdigest()

    try:

        public_key_client = client_socket.recv(4096)
        print("\nChave pública do cliente recebida.")

        # Envia a chave pública do servidor para o cliente
        client_socket.send(public_key_server)
        print("Chave pública do servidor enviada.")

        # Recebe a chave AES criptografada
        encrypted_aes_key = client_socket.recv(4096)
        aes_key = decrypt_aes_key(encrypted_aes_key)
        print("Chave AES recebida e descriptografada.")

        # Agora pode receber mensagens criptografadas com AES
        while True:
            encrypted_message = client_socket.recv(4096)
            if not encrypted_message:
                break

            # Descriptografa a mensagem
            decrypted_message = decrypt_message(encrypted_message)
            if decrypted_message is None:
                print("\nErro: Mensagem não pôde ser descriptografada.")
                continue

            # Recebe o hash da mensagem
            received_hash = client_socket.recv(4096).decode('utf-8')

            # Calcula o hash da mensagem descriptografada
            calculated_hash = calculate_hash(decrypted_message)

            # Verifica a integridade
            if received_hash == calculated_hash:
                print(f"\nMensagem recebida (íntegra): {decrypted_message}")
            else:
                print(f"\nMensagem recebida (corrompida): {decrypted_message}")
    except Exception as e:
        print(f"\nErro ao receber mensagem: {e}")
    finally:
        client_socket.close()

def send_messages(client_socket):
    try:

from Crypto.PublicKey import RSA
from Crypto.Cipher import PKCS1_OAEP, AES
from Crypto.Hash import SHA256
from Crypto.Random import get_random_bytes
import socket
import threading

# Gera um par de chaves RSA para o cliente
key_pair_client = RSA.generate(2048)
private_key_client = key_pair_client.export_key()
public_key_client = key_pair_client.publickey().export_key()

# Variável para armazenar a chave simétrica (AES)
aes_key = None

def encrypt_aes_key(public_key_server):
    cipher = PKCS1_OAEP.new(RSA.import_key(public_key_server))
    return cipher.encrypt(aes_key)

def encrypt_message(message):
    cipher = AES.new(aes_key, AES.MODE_EAX)
    nonce = cipher.nonce
    ciphertext, tag = cipher.encrypt_and_digest(message.encode('utf-8'))
    return nonce + ciphertext + tag

def decrypt_message(encrypted_message):
    nonce = encrypted_message[:16]
    ciphertext = encrypted_message[16:-16]
    tag = encrypted_message[-16:]
    cipher = AES.new(aes_key, AES.MODE_EAX, nonce=nonce)
    return cipher.decrypt_and_verify(ciphertext, tag).decode('utf-8')

def calculate_hash(message):
    return SHA256.new(message.encode('utf-8')).hexdigest()

def receive_messages(client_socket):
    global aes_key
    try:
        # Envia a chave pública do cliente para o servidor
        client_socket.send(public_key_client)
        print("\nChave pública do cliente enviada.")

        # Recebe a chave pública do servidor
        public_key_server = client_socket.recv(4096)
        print("Chave pública do servidor recebida.")

        # Gera uma chave AES e a criptografa com a chave pública do servidor
        aes_key = get_random_bytes(16)
        encrypted_aes_key = encrypt_aes_key(public_key_server)
        client_socket.send(encrypted_aes_key)
        print("Chave AES enviada e criptografada.")

        # Agora pode receber mensagens criptografadas com AES
        while True:
            encrypted_message = client_socket.recv(4096)
            if not encrypted_message:
                break

            # Descriptografa a mensagem
            decrypted_message = decrypt_message(encrypted_message)
            if decrypted_message is None:
                print("\nErro: Mensagem não pôde ser descriptografada.")
                continue

            # Recebe o hash da mensagem
            received_hash = client_socket.recv(4096).decode('utf-8')

            # Calcula o hash da mensagem descriptografada
            calculated_hash = calculate_hash(decrypted_message)

            # Verifica a integridade
            if received_hash == calculated_hash:
                print(f"\nMensagem recebida (íntegra): {decrypted_message}")
            else:
                print(f"\nMensagem recebida (corrompida): {decrypted_message}")
    except Exception as e:
        print(f"\nErro ao receber mensagem: {e}")
    finally:
        client_socket.close()

def send_messages(client_socket):
    try:
        while True:
            message = input("\nDigite uma mensagem para enviar: ")
            if aes_key:
                # Criptografa a mensagem
                encrypted_message = encrypt_message(message)

                message_hash = calculate_hash(message)

                print("\nAguardando chave AES...")

4. Explicação dos Conceitos Técnicos

Socket

O que é?: Um socket é um ponto de comunicação entre dois processos em uma rede. Ele permite que programas se comuniquem, seja na mesma máquina ou em máquinas diferentes.

Uso no Projeto: Usamos sockets para estabelecer uma conexão TCP entre o servidor e o cliente.

Threading

O que é?: Threading é uma técnica que permite a execução simultânea de múltiplas tarefas em um programa.

Uso no Projeto: Usamos threads para permitir que o servidor envie e receba mensagens simultaneamente.

Vetor de Inicialização (IV)

O que é?: O IV é um valor aleatório usado junto com a chave para criptografar os dados. Ele garante que, mesmo que a mesma mensagem seja criptografada várias vezes, o resultado será diferente a cada vez.

Uso no Projeto: No modo CBC do AES, o IV é usado para aumentar a segurança da criptografia.

Modo CBC (Cipher Block Chaining)

O que é?: O modo CBC é um modo de operação para criptografia de blocos. Ele divide a mensagem em blocos e criptografa cada bloco em sequência, usando o resultado do bloco anterior.

Uso no Projeto: Usamos o modo CBC no AES para garantir que blocos idênticos na mensagem original resultem em blocos criptografados diferentes.

Cipher

O que é?: Um cipher é um algoritmo de criptografia que transforma dados legíveis (plaintext) em dados ilegíveis (ciphertext) e vice-versa.

Uso no Projeto: Usamos ciphers como AES e RSA para criptografar e descriptografar mensagens.

UTF-8

O que é?: UTF-8 é um padrão de codificação de caracteres que permite representar textos em diferentes idiomas. Ele é compatível com ASCII e suporta caracteres especiais e emojis.

Uso no Projeto: Codificamos as mensagens em UTF-8 antes de criptografá-las e as decodificamos após descriptografá-las.

Tratamento de Erros e Exceptions

O que é?: O tratamento de erros é uma técnica usada para lidar com situações inesperadas durante a execução de um programa. Em Python, isso é feito com blocos try-except.

Uso no Projeto: Usamos tratamento de erros para capturar exceções que possam ocorrer durante a comunicação, como falhas na conexão ou erros de criptografia.

Funções de Hash (SHA-256)

O que é?: Uma função de hash pega uma entrada de qualquer tamanho e gera um valor de tamanho fixo, chamado de hash. O SHA-256 é uma função de hash amplamente utilizada, que gera um hash de 256 bits (32 bytes).

Uso no Projeto: Usamos o SHA-256 para garantir a integridade das mensagens. O hash é calculado antes de enviar a mensagem e verificado após o recebimento.

Nonce (Número Usado Uma Vez)

O que é?: Um nonce é um valor aleatório usado em criptografia para garantir que a mesma mensagem criptografada várias vezes resulte em textos cifrados diferentes.

Uso no Projeto: No modo EAX do AES, o nonce é usado para aumentar a segurança da criptografia.

Autenticação e Integridade (Tag)

O que é?: A autenticação e a integridade são garantidas por meio de uma tag, que é um valor gerado durante a criptografia e verificado durante a descriptografia.

Uso no Projeto: No modo EAX do AES, a tag é usada para garantir que a mensagem não foi alterada durante a transmissão.
